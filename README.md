# рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ 1 Рђћ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў (Variables)

рЃўрЃАрЃгрЃљрЃЋрЃџрЃћ рЃарЃЮрЃњрЃЮрЃа рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў JavaScript-рЃерЃў.

## ­ЪЊџ рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃЏрЃўрЃќрЃюрЃћрЃЉрЃў

рЃљрЃЏ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃўрЃњрЃћрЃЉ:

- рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃюрЃЋрЃАрЃљрЃќрЃдрЃЋрЃарЃЮрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў `let`-рЃўрЃЌ
- рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљрЃА рЃАрЃбрЃарЃўрЃюрЃњрЃАрЃљ рЃЊрЃљ рЃюрЃЮрЃЏрЃћрЃарЃА рЃерЃЮрЃарЃўрЃА
- рЃАрЃбрЃарЃўрЃюрЃњрЃћрЃЉрЃўрЃА рЃерЃћрЃћрЃарЃЌрЃћрЃЉрЃљрЃА `+` рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃўрЃЌ
- рЃарЃЮрЃњрЃЮрЃа рЃљрЃЋрЃўрЃарЃЕрЃўрЃЮрЃЌ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃгрЃЮрЃарЃў рЃфрЃЋрЃџрЃљрЃЊрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў

---

## РюЇ№ИЈ рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃў

рЃерЃћрЃљрЃАрЃарЃБрЃџрЃћ рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў:

1. **рЃерЃћрЃЦрЃЏрЃћрЃюрЃў `firstName` рЃфрЃЋрЃџрЃљрЃЊрЃў** Рђћ рЃерЃћрЃюрЃў рЃАрЃљрЃ«рЃћрЃџрЃў (string)
2. **рЃерЃћрЃЦрЃЏрЃћрЃюрЃў `lastName` рЃфрЃЋрЃџрЃљрЃЊрЃў** Рђћ рЃерЃћрЃюрЃў рЃњрЃЋрЃљрЃарЃў (string)
3. **рЃерЃћрЃЦрЃЏрЃюрЃў `fullName` рЃфрЃЋрЃџрЃљрЃЊрЃў** Рђћ рЃЊрЃљрЃљрЃћрЃарЃЌрЃћ firstName рЃЊрЃљ lastName рЃАрЃўрЃЋрЃарЃфрЃўрЃЌ (space-рЃўрЃЌ)
   - рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: `"John" + " " + "Doe"` = `"John Doe"`
4. **рЃерЃћрЃЦрЃЏрЃюрЃў `age` рЃфрЃЋрЃџрЃљрЃЊрЃў** Рђћ рЃАрЃарЃБрЃџрЃгрЃџрЃЮрЃЋрЃљрЃюрЃў рЃљрЃАрЃљрЃЎрЃў 1-рЃЊрЃљрЃю 150-рЃЏрЃЊрЃћ (number)
5. **рЃерЃћрЃЦрЃЏрЃюрЃў `city` рЃфрЃЋрЃџрЃљрЃЊрЃў** Рђћ рЃерЃћрЃюрЃў рЃЦрЃљрЃџрЃљрЃЦрЃў (string)
6. **рЃерЃћрЃЦрЃЏрЃюрЃў `output` рЃфрЃЋрЃџрЃљрЃЊрЃў** Рђћ рЃбрЃћрЃЦрЃАрЃбрЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃЌ:
   ```
   "My name is [fullName]. I am [age] years old and I live in [city]."
   ```
   - рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: `"My name is John Doe. I am 25 years old and I live in Tbilisi."`

### Рџа№ИЈ рЃгрЃћрЃАрЃћрЃЉрЃў:

- РюЌ рЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў (functions)
- РюЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ `let` рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃўрЃАрЃЌрЃЋрЃўрЃА
- РюЊ рЃАрЃбрЃарЃўрЃюрЃњрЃћрЃЉрЃўрЃА рЃерЃћрЃћрЃарЃЌрЃћрЃЉрЃљ рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћ `+` рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃўрЃЌ
- РюЊ рЃДрЃЋрЃћрЃџрЃљ рЃфрЃЋрЃџрЃљрЃЊрЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃБрЃфрЃљрЃарЃўрЃћрЃџрЃў (рЃљрЃарЃљрЃЋрЃўрЃЌрЃљрЃарЃў "")

рЃЎрЃЮрЃЊрЃў рЃгрЃћрЃарЃћ **`src/index.js`** рЃцрЃљрЃўрЃџрЃерЃў.

## ­Ъџђ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

```bash
# рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ (рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃљрЃА)
npm install

# рЃбрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ
npm test

# рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃбрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ (рЃарЃЮрЃЏрЃћрЃџрЃў рЃбрЃћрЃАрЃбрЃў рЃарЃљрЃбрЃЮрЃЏ рЃЕрЃљрЃўрЃГрЃарЃљ)
npm run test:verbose
```

## РюЁ рЃЏрЃЮрЃАрЃљрЃџрЃЮрЃЊрЃюрЃћрЃџрЃў рЃерЃћрЃЊрЃћрЃњрЃў

рЃДрЃЋрЃћрЃџрЃљ рЃбрЃћрЃАрЃбрЃў рЃБрЃюрЃЊрЃљ **PASS**-рЃљрЃЊ рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃћрЃА (рЃАрЃБрЃџ 14 рЃбрЃћрЃАрЃбрЃў):

**рЃАрЃбрЃарЃўрЃюрЃњ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў (3 рЃбрЃћрЃАрЃбрЃў):**

- РюЊ `firstName` рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃБрЃфрЃљрЃарЃўрЃћрЃџрЃў рЃАрЃбрЃарЃўрЃюрЃњрЃў
- РюЊ `lastName` рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃБрЃфрЃљрЃарЃўрЃћрЃџрЃў рЃАрЃбрЃарЃўрЃюрЃњрЃў
- РюЊ `city` рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃБрЃфрЃљрЃарЃўрЃћрЃџрЃў рЃАрЃбрЃарЃўрЃюрЃњрЃў

**`fullName` рЃфрЃЋрЃџрЃљрЃЊрЃў (4 рЃбрЃћрЃАрЃбрЃў):**

- РюЊ `fullName` рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃБрЃфрЃљрЃарЃўрЃћрЃџрЃў рЃАрЃбрЃарЃўрЃюрЃњрЃў
- РюЊ `fullName` рЃБрЃюрЃЊрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА `firstName`-рЃА
- РюЊ `fullName` рЃБрЃюрЃЊрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА `lastName`-рЃА
- РюЊ `fullName`-рЃерЃў рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃАрЃўрЃЋрЃарЃфрЃћ firstName-рЃАрЃљ рЃЊрЃљ lastName-рЃА рЃерЃЮрЃарЃўрЃА

**`age` рЃфрЃЋрЃџрЃљрЃЊрЃў (3 рЃбрЃћрЃАрЃбрЃў):**

- РюЊ `age` рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃарЃўрЃфрЃ«рЃЋрЃў
- РюЊ `age` рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА 0-рЃќрЃћ рЃЏрЃћрЃбрЃў
- РюЊ `age` рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА 150-рЃќрЃћ рЃюрЃљрЃЎрЃџрЃћрЃЉрЃў рЃљрЃю рЃбрЃЮрЃџрЃў

**`output` рЃфрЃЋрЃџрЃљрЃЊрЃў (4+ рЃбрЃћрЃАрЃбрЃў):**

- РюЊ `output` рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃБрЃфрЃљрЃарЃўрЃћрЃџрЃў рЃАрЃбрЃарЃўрЃюрЃњрЃў
- РюЊ `output` рЃБрЃюрЃЊрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА `fullName`-рЃА
- РюЊ `output` рЃБрЃюрЃЊрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА `age`-рЃА
- РюЊ `output` рЃБрЃюрЃЊрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА `city`-рЃА
- РюЊ `output` рЃБрЃюрЃЊрЃљ рЃўрЃгрЃДрЃћрЃЉрЃЮрЃЊрЃћрЃА рЃбрЃћрЃЦрЃАрЃбрЃўрЃЌ "My name is"
- РюЊ `output` рЃБрЃюрЃЊрЃљ рЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃЮрЃЊрЃћрЃА рЃгрЃћрЃарЃбрЃўрЃџрЃўрЃЌ

## ­ЪњА рЃарЃЕрЃћрЃЋрЃћрЃЉрЃў

### рЃАрЃбрЃарЃўрЃюрЃњрЃћрЃЉрЃўрЃА рЃерЃћрЃћрЃарЃЌрЃћрЃЉрЃљ

рЃАрЃбрЃарЃўрЃюрЃњрЃћрЃЉрЃўрЃА рЃћрЃарЃЌрЃљрЃЊ рЃерЃћрЃЎрЃЋрЃарЃљ рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ `+` рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃўрЃЌ рЃ«рЃЊрЃћрЃЉрЃљ:

```javascript
let greeting = "Hello" + " " + "World"; // "Hello World"
```

### рЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃАрЃбрЃарЃўрЃюрЃњрЃљрЃЊ рЃЕрЃљрЃАрЃЏрЃљ

рЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃАрЃбрЃарЃўрЃюрЃњрЃерЃў рЃЕрЃљрЃАрЃЏрЃўрЃАрЃљрЃА JavaScript рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА рЃЏрЃљрЃА:

```javascript
let age = 25;
let text = "I am " + age + " years old"; // "I am 25 years old"
```

### рЃ«рЃерЃўрЃарЃў рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃў

- РЮї рЃЉрЃарЃГрЃДрЃљрЃџрЃћрЃЉрЃўрЃА рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ: `let x = "He said "hello"";`
- РЮї рЃАрЃўрЃЋрЃарЃфрЃўрЃА рЃЊрЃљрЃЋрЃўрЃгрЃДрЃћрЃЉрЃљ рЃерЃћрЃћрЃарЃЌрЃћрЃЉрЃўрЃАрЃљрЃА: `"John" + "Doe"` Рєњ `"JohnDoe"`
- РЮї рЃљрЃАрЃљрЃЎрЃўрЃА рЃАрЃбрЃарЃўрЃюрЃњрЃљрЃЊ рЃЊрЃљрЃгрЃћрЃарЃљ: `let age = "25";` (рЃБрЃюрЃЊрЃљ рЃўрЃДрЃЮрЃА рЃарЃўрЃфрЃ«рЃЋрЃў)

---

## ­Ъј» рЃЉрЃЮрЃюрЃБрЃАрЃў (рЃљрЃарЃљрЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ)

рЃАрЃфрЃљрЃЊрЃћ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ:

```javascript
let hobby = ""; // рЃерЃћрЃюрЃў рЃ░рЃЮрЃЉрЃў
let hobbySentence = ""; // рЃЏрЃљрЃњ.: "My hobby is reading"
```

рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃљрЃгрЃДрЃћ рЃћрЃарЃЌрЃў рЃАрЃарЃБрЃџрЃў рЃгрЃўрЃюрЃљрЃЊрЃљрЃЊрЃћрЃЉрЃљ:

```javascript
// рЃЏрЃљрЃњ.: "My name is John Doe. I am 25 years old, I live in Tbilisi and my hobby is reading."
let fullBio = "";
```

## ­ЪЊЮ рЃЕрЃљрЃЉрЃљрЃарЃћрЃЉрЃљ

рЃАрЃљрЃюрЃљрЃЏ рЃЕрЃљрЃљрЃЉрЃљрЃарЃћрЃЉ:

1. рЃњрЃљрЃБрЃерЃЋрЃў рЃбрЃћрЃАрЃбрЃћрЃЉрЃў: `npm test`
2. рЃерЃћрЃўрЃюрЃљрЃ«рЃћ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў: `git commit -m "Variables рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃљ"`
3. рЃљрЃбрЃЋрЃўрЃарЃЌрЃћ GitHub-рЃќрЃћ: `git push`

Autograding рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃњрЃўрЃцрЃўрЃЦрЃАрЃўрЃарЃћрЃЉрЃА рЃЦрЃБрЃџрЃћрЃЉрЃА! ­ЪјЊ

## ­Ъћј рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЋрЃўрЃњрЃЮ рЃарЃљрЃбрЃЮрЃЏ рЃЕрЃљрЃўрЃГрЃарЃљ рЃбрЃћрЃАрЃбрЃў

рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ рЃњрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃљрЃЊ рЃарЃЮрЃЏрЃћрЃџрЃў рЃбрЃћрЃАрЃбрЃў рЃЕрЃљрЃЋрЃљрЃарЃЊрЃљ рЃЊрЃљ рЃарЃљ рЃўрЃДрЃЮ рЃЏрЃЮрЃАрЃљрЃџрЃЮрЃЊрЃюрЃћрЃџрЃў:

```bash
npm run test:verbose
```

рЃарЃЮрЃњрЃЮрЃа рЃгрЃљрЃЋрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃЌ рЃерЃћрЃЊрЃћрЃњрЃў:

- рЃбрЃћрЃАрЃбрЃћрЃЉрЃў рЃЊрЃљрЃ»рЃњрЃБрЃцрЃћрЃЉрЃБрЃџрЃўрЃљ (рЃЏрЃљрЃњ.: "String Variables", "fullName Variable")
- рЃЕрЃљрЃЋрЃљрЃарЃЊрЃюрЃўрЃџрЃў рЃбрЃћрЃАрЃбрЃў рЃњрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА:
  - рЃарЃљ рЃўрЃДрЃЮ рЃЏрЃЮрЃАрЃљрЃџрЃЮрЃЊрЃюрЃћрЃџрЃў (Expected)
  - рЃарЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃЏрЃўрЃўрЃдрЃћрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃЏрЃљ (Received)
  - рЃарЃЮрЃЏрЃћрЃџ рЃ«рЃљрЃќрЃќрЃћрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃљ `tests/variables.test.js`-рЃерЃў

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ:

```
Expected: > 0
Received:   0
```

рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃфрЃЋрЃџрЃљрЃЊрЃў рЃфрЃљрЃарЃўрЃћрЃџрЃўрЃљ рЃЊрЃљ рЃБрЃюрЃЊрЃљ рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃЮ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ.
